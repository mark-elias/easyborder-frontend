# Live Web-App: 
https://easyborder.io/

About: EasyBorder provides reliable border wait times using official CBP data for all port of entries and crossing methods. Compare crossings, save your favorite lanes, and share your border crossing experiences with other travelers.

Frontend: Typescript, Next.js, Tailwind, Shadcn, Bun, Tanstack Query, React-Hook-Form, Zod, Zustand

Backend: TypeScript, Nest.js

Auth: Passport.js, jwts, cookies

Infra: AWS EC2, Nginx, Docker Compose, Github Actions
